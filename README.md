## 🎯 Overview

Ever downloaded a video course (Udemy, Coursera, etc.) and wanted to quickly search through all the transcripts? This PowerShell script automatically processes hundreds of subtitle files (`.vtt` and `.srt`) and converts them into clean, organized, and searchable text files.

**Perfect for:**
- 📚 Course study notes and reference
- 🔍 Quick topic searching across entire courses
- 📝 Creating study guides from video content
- 🎓 Academic research and note-taking
- 💼 Professional development tracking

## ✨ Features

### 🧹 **Smart Cleaning**
- Removes timestamps (`00:01:23.456 --> 00:01:26.789`)
- Strips metadata (`WEBVTT`, `NOTE`, `STYLE`)
- Eliminates sequence numbers and empty lines
- Filters out sound effects (`[music]`, `[applause]`)

### 📁 **Organized Output**
- Maintains course folder structure
- Creates individual clean transcript files
- Generates combined master file for searching
- Handles special characters and emojis in filenames

### 🎯 **User-Friendly**
- Progress tracking with colored output
- Comprehensive error handling
- Automatic results folder opening
- Detailed processing statistics

## 🔧 Requirements

- **Windows 10/11** or Windows Server
- **PowerShell 5.1** or higher
- **Course folder** with `.vtt` or `.srt` subtitle files

Network drives: Copy to local drive first for faster processing
